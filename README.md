# Gecko Migration Status

This document tracks the implementation status of features as I migrate from WebView to GeckoView. WebView will remain supported as the default engine, while GeckoView will be available as an alternative build flavor.

## Feature Implementation Status

| Feature | WebView | GeckoView | Description |
|---------|:-------:|:---------:|-------------|
| Launch | ✅ | ❌ | Basic app launch functionality |
| Load Apps | ✅ | ❌ | Loading application list |
| Load Assets | ✅ | ❌ | Loading app resources |
| Settings | ✅ | ❌ | Groove Settings interface |
| Tweaks | ✅ | ❌ | Groove Tweaks interface |
| App Launch | ✅ | ❌ | Application execution |

### Legend
- ✅ Implemented
- ❌ Not implemented
